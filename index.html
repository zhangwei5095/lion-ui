Lion Framework

Lion Framework是一个基于Spring MVC、Spring、Hibernate、Shiro、Bootstrap的基础开发框架，大大提升了开发速度。
主要组件有： UI：datagrids、combo、combotree、form扩展、对话框架；
数据组件：data；
常用工具： common API serivce dsession 分布式会话 redis 基于Redis读写操作 monitor 简单监控模块；
