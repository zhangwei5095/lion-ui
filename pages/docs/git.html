Git下载及安装

Git下载地址

下载地址：http://maven.apache.org/download.cgi

  1.在Downloads下选择Windows

  2.在此下载

Git安装及配置

1.点击下一步

2.继续点击下一步

3.选择Git的安装路径

4.在Windows Explorer integration选项中将“Git Bash here”和“Git GUI here”打对勾。

5.点击下一步

6.在“Adjusting your PATH environment”选项中，默认即可。

7.在“Configuring the line ending conversions”选项中，选择第一个选项：如果是跨平台项目，在windows系统安装。

8.安装完成，并可以在桌面上看到Git Bash的图标
