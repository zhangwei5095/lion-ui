说明

单选按钮每次只能选择一项，当选择一项再选其他项时，之前的选择自动消失。

前置条件

页面引用JS与CSS文件

<!--整个框架依赖的JS文件-->
<script src="../../src/js/zepto.min.js"></script>
<!--当前组件依赖的JS文件-->
<script src="../../src/js/one.js"></script>
<!--当前组件JS文件-->
<script src="../../src/js/ui/radioButton/radioButton.js"></script>
<!--当前组件依赖的CSS文件-->
<link href="../../src/css/one.css" rel="stylesheet">
<!--当前组件CSS文件-->
<link href="../../src/css/ui/radioButton.css" rel="stylesheet">

使用方式

1.定义HTML

<label class="one_radio-button radio">
    <input type="radio" name="r" checked="">
    <div class="one_radio-button__checkmark"></div>
    <span>text</span>
</label>
<label class="one_radio-button radio">
    <input type="radio" name="r" checked=false>
    <div class="one_radio-button__checkmark"></div>
    <span>enable</span>
</label>
<label class="one_radio-button radio">
    <input type="radio" name="r"  >
    <div class="one_radio-button__checkmark"></div>
    <span>enable</span>
</label>
<label class="one_radio-button radio">
    <input type="radio" name="r" value="val" disabled="disabled" >
    <div class="one_radio-button__checkmark"></div>
    <span>enable</span>
</label>

2.使用JavaScript调用

    $(function(){
       //初始化组件
        var radio1= one.ui.radioButton(".radio");
     })

高级使用

1.设置选中状态

    $(function(){
       //初始化组件
        var radio1= one.ui.radioButton(".radio");
     
       //设置第四个选中
        radio1.setCheck(3,true);
     
       //设置第四个有效
        radio1.setDisable(3,false);//索引是3
     
       //设置第四个无效同时第一个选中
        radio1.setDisable(3,true).setCheck(0,true);
     })
