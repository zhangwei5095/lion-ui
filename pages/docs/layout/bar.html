说明

通常用于显示头部或底部的导航栏。

前置条件

页面引用JS与CSS文件

<!--整个框架依赖的JS文件-->
<!--当前组件依赖的CSS文件-->
<link href="../../src/css/one.css" rel="stylesheet">
<link href="../../src/css/layout/bar.css" rel="stylesheet">

使用方式

1.用法一

简单的标题: 
<div class="one-bar">
    <ul class="one-g">

        <li class="one-u-sm-12 no-active">
            登&nbsp;&nbsp;录
        </li>

    </ul>
</div>

2.用法二

三列结构，固定在顶部 
   <div class="one-bar one-bar-fixed-top">
    <ul class="one-g">
        <li class="one-u-sm-2">
            返回
        </li>
        <li class="one-u-sm-8 no-active">
            登&nbsp;&nbsp;录
        </li>
        <li class="one-u-sm-2">
            关闭
        </li>
    </ul>
</div>

3.用法三

固定在底部 
<div class="one-bar one-bar-fixed-bottom">
    <ul class="one-g">

        <li class="one-u-sm-12 no-active">
            固定底部
        </li>

    </ul>
</div>
