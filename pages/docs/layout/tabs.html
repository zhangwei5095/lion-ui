说明

以Tab的方式展示。

前置条件

页面引用JS与CSS文件

<!--当前组件依赖的CSS文件-->
<link href="../../src/css/one.css" rel="stylesheet">

使用方式

1.用法一

<div class="one-tabs  one-tabs-top">
    <a class="one-tab-item">
        主页
    </a>
    <a class="one-tab-item">
        收藏
    </a>
    <a class="one-tab-item">
        设置
    </a>
</div>

2.用法二

带图标,需要引用对应的iconfont（字体样式文件） 
<div class="one-tabs tabs1">
    <a class="one-tab-item">
        <i class="icon ion-home"></i>
        主页
    </a>
    <a class="one-tab-item">
        <i class="icon ion-star"></i>
        收藏
    </a>
    <a class="one-tab-item">
        <i class="icon ion-gear-a"></i>
        设置
    </a>
</div>
