Git项目加载

目录

  • 1在视图中打开git Repositories
  • 2.克隆git项目到本地
  • 3.导入项目

1在视图中打开git Repositories

进入eclipse，在上方工具栏打开windows->Show View->Other，选中Git Repositories并点击ok。

2.克隆git项目到本地

  • 点击Clone a Git repository 在URI中写入https://github.com/scwanglijun/framework/网址，并输入用户名和密码。

  • 在Directory中选中对应的目录，并点击Finish结束，等待项目下载完成。

3.导入项目

将项目作为Maven Project导入。
