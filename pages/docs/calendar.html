说明

显示日历，可以调节显示当前时间，也可以显示指定日期时间。

前置条件

页面引用JS与CSS文件

<!--整个框架依赖的JS文件-->
<script src="../../src/js/zepto.min.js"></script>
<!--当前组件依赖的JS文件-->
<script src="../../src/js/one.js"></script>
<!--当前组件JS文件-->
<script src="../../src/js/ui/calendar/calendar.js"></script>
<!--当前组件依赖的CSS文件-->
<link href="../../src/css/one.css" rel="stylesheet">
<!--当前组件CSS文件-->
<link href="../../src/css/ui/calendar.css" rel="stylesheet">

使用方式

1.定义HTML

<div class="one_ui_content">
    <input type="text" readonly class="one_ui_input" id="timedata" maxlength="20" placeholder="日期" name="timedata" value="">
</div>

2.使用JavaScript调用

    $(function(){
       //初始化组件
        window.demo = $('#timedata').mobiscroll('destroy');
     })

高级使用

1.指定默认的日期

     $(function(){
    //初始化组件
    window.demo = $('#timedata').mobiscroll('destroy');
    demo.date({
        //preset : 'date',  指定显示类型（当使用mobiscroll（）调用时）
        dateOrder : 'yymmdd',//滚轮中的顺序
        dateFormat : 'yyyy-mm-dd',//input 中的格式
        theme : 'android-ics light',
        mode : 'scroller', //mixed///clickpick
        display : 'bottom',
        lang : 'zh',
        defaultValue : new Date(1998, 11, 13)  //指定默认的日期
    });
    demo.mobiscroll('show');
})
