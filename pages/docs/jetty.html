Jetty安装

Jetty —是一个Java EE服务器，Jetty有专门的Eclipse插件叫做：run-jetty-run，按照Eclipse安装插件的标准步骤：

1.Help -> Install New Software

2.在Work with中输入：http://run-jetty-run.googlecode.com/svn/trunk/updatesite
会出现2个RunJettyRun，建议选择1.3.2版本，根据个人喜好。

3.Eclipse会检查，一些前置依赖，没有它会自动下载，下载完成之后如下图。

4.继续Next（下一步）…

5.点击Finish开始安装

6.安装完成之后，会提示重启Eclipse，直接重启就好了。
