1.js库

<script src="${base}/resources/global/js/combotree/combotree.js" type="text/javascript"></script>

2.css库

<link href="${base}/resources/global/css/combotree/combotree.css" rel="stylesheet" type="text/css" />

使用方法

1.js

//获取选中的值
 Var data = $(‘#departmentId’).val();

2.HTML

<input  id="departmentId"  name="departmentId"  placeholder="请选择部门…"
type="text"  class="lion-combotree form-control"
data-loadURL="${base}/system/department/comboxtree.json"
data-width="200px" data-height="300px"/>

属性

名称 描述
id 规定该标签的唯一id。
name 规定该标签的名称。
placeholder 规定如果该标签的值为空时，显示的内容。
class 规定该标签的样式。推荐样式（” tdon-combotree form-control”)
data-width 规定该标签树形下拉框的宽度。
data-height 规定该标签树形下拉框的高度。
data-URL 规定该标签获取数据的url。

方法

名称 参数 描述
setTree none 设置树的数据结构。
val Id 获取数据或设置选中的数据
treeOnCtdck e, treeId, treeNode 当在树形下拉框中选择数据时触发事件
treeBeforeCtdck treeId, treeNode 当在树形下拉框中选择数据之前触发事件
