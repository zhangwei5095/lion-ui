使用方法

1.导航栏

//加载加载导航栏
lion.util.menu();

2.提示

//成功提示
lion.util.success(‘提示’,‘成功’);
//错误提示
lion.util.error(‘提示’,‘错误’);
//警告提示
lion.util.warning(‘提示’,‘警告’);
//提示
lion.util.info(提示’,‘请选择要编辑的记录’);

3.数据提交

html:
<form id="sysForm">
    <input type="text" id="name" name="username"/>
</form>
js:
var param={'username':'demo'};
lion.web.post({url:'add.json',data:param,success:successFrm});
function successFrm(result,args){
location.reload();}

方法

名称 参数 描述
tdon.util.isEmpty o 判断对象是否为空
tdon.util.isNotEmpty o 判断对象是否不为空
tdon.util.context none 获取应用上下文根属性
tdon.util.hostname none 获取域名
tdon.util.protocol none 获取当前网络协议
tdon.util.httpurl none 获取当前路径
tdon.util.menu none 加载导航栏
tdon.util.success title,content 成功提示框
tdon.util.error title,content 错误提示框
tdon.util.warning title,content 警告提示框
tdon.util.info title,content 提示框
tdon.util.post url,data,successfn,errorfn,arg Ajax post请求(错误方法不传则使
用默认方法）
tdon.util.postasync url,data,successfn,errorfn,arg Ajax post同步请求(错误方法不传
则使用默认方法）
tdon.util.postjson url,data,successfn,errorfn,arg Ajax post json请求(错误方法不传
则使用默认方法）
tdon.util.get url,successfn,errorfn,arg Ajax get请求(错误方法不传则使
用默认方法）
