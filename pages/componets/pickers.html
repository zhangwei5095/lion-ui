        todayBtn:  1,
        autoclose: 1,
        todayHighlight: 1,
        startView: 2,
        forceParse: 0,
        showMeridian: 1
    });

属性

名称 描述
format 日期格式， p, P, h, hh, i, ii, s, ss, d, dd, m, mm, M, MM, yy, yyyy 的任意组合。
weekStart 一周从哪一天开始。0（星期日）到6（星期六）
autoclose 当选择一个日期之后是否立即关闭此日期时间选择器。
startView 日期时间选择器打开之后首先显示的视图。
minView 日期时间选择器所能够提供的最精确的时间选择视图。
maxView 日期时间选择器最高能展示的选择范围视图。
forceParse 当选择器关闭的时候，是否强制解析输入框中的值。

方法

名称 参数 描述
remove none 移除日期时间选择器。同时移除已经绑定的event、内部绑定的对象和HTML元素。
show none 显示日期时间选择器。
update none 使用当前输入框中的值更新日期时间选择器。
setStartDate startDate (String) 给日期时间选择器设置一个新的起始日期。
setEndDate endDate (String) 给日期时间选择器设置结束日期。
