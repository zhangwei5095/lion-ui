属性

名称 描述
id 规定该数据表的唯一id。
name 规定该数据表的名称。
class 规定该数据表的样式。推荐样式（”tdon-datagrids
table table-striped table-bordered table-hover”）
data-singleselect 规定该数据表的勾选框是否单选
data-checkbox 规定该数据表是否使用勾选框
data-URL 规定该数据表获取数据的url。
data-pageSize 规定该数据表每次展出数据的行数
data-field 规定该列展示的对象的属性
data-sortable 规定该列是否可以进行排序
data-formatter 规定该列进行数据转换的方法

方法

名称 参数 描述
getSelections none 多行数据返回已选择的数据
getSelected none 获取选择的行数据row
getdata none 获取所有数据
createdRow row, data, index 创建行回调函数
reload none 重新加载数据
querydata none 获取查询参数
initComplete none 初始化事件
loadinitComplete none DataTable初始化事件
