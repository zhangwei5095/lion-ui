使用方法

1.js

//通过方法来获取combo的数据或对combo进行操作
var data = $(‘iconTypeList’).combo(‘getData’);

2.HTML

 <select  id="iconTypeList" name="iconType"  placeholder请输入图标..."
    class="lion-combo form-control select2" data-valueField='codeValue'
    data-textField='nameZh' data-URL="${base}/system/code/combox.htm?
    nameEn=IconType"></select>

属性

名称 描述
id 规定该标签的唯一id。
name 规定该标签的名称。
placeholder 规定如果该标签的值为空时，显示的内容。
class 规定该标签的样式。推荐样式（”tdon-combo form-control select2”）
data-valueField 规定该标签下拉框每一个元素的值所对应的对象的属性。
data-textField 规定该标签下拉框每一个元素显示的值所对应的对象的属性。
data-URL 规定该标签获取数据的url。

方法

名称 参数 描述
getVal none 获取默认选择的数据。
setVal param 将用户选中的元素设置值。
clear none 清除先前的选择。
getData none 获取所有数据。
